🌾 Kasan Yar - Pakistani AI Chatbot

Kasan Yar is an intelligent offline Pakistani AI chatbot developed in Python. It uses a custom dataset containing more than 5,000 question-answer pairs and over 21,500 lines of knowledge to provide useful information without relying on external APIs or online services.

✨ Features

- 🌱 Agriculture and crop information
- 🌦️ Weather-related guidance
- 🖼️ Image information support
- 💊 Medicine and daily life information
- 🤖 Smart response generation
- 📚 Custom dataset with 5,000+ Q&A pairs
- 🔌 Fully offline chatbot
- ⚡ Lightweight and efficient
- 🐍 Developed using Python
